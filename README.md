# DEMO 3D shooter

Third person shooter demo. (smells like shit but it's almost playable)
To pass a level you have to shoot all enemies. You have to find and pick up all ammo, you will need that.
Each gun has it's type of ammo.

Left click - Shoot
Right click - Aim
WASD - Move
Spacebar - Jump
Shift - Run
1, 2, 3 - Weapon switch
